# Vinea

Modular autonomous robot platform for Dutch commercial greenhouse operators.
Harvesting + scouting on one configurable base. Works in existing infrastructure — no rebuild required.
Sold as a monthly service (RaaS).

> **Status:** Pre-prototype. Idea anatomy complete. Currently in validation phase — talking to real growers before building anything.

---

## The Problem

Dutch greenhouse operators spend ~€1.6M/year per 5-hectare site on manual harvesting and scouting labour. Seasonal labour supply is structurally collapsing. The only funded competitor (SAIA Agrobotics) requires a full greenhouse rebuild — leaving ~95% of the market with no viable automation option.

## The Solution

A modular robot that works in existing greenhouse row infrastructure:

- **Harvest module** — autonomous picking + de-leafing, CV-guided, crop-specific grippers (tomato first)
- **Scout module** — daily plant health inspection, disease/pest detection 3–5 days before visible symptoms, grower dashboard

Same chassis across all customers. Only the gripper, row-width adapter, and module set vary.

## Business Model

**Robot-as-a-Service (RaaS)** — monthly subscription covering hardware, software, maintenance, and data.

| Tier | Includes | Price/robot/month |
|---|---|---|
| Harvest | Picking + de-leafing | €4,500 |
| Harvest + Scout | + scouting + dashboard | €5,800 |
| Full Analytics | + yield forecasting, agronomist insights | €6,500 |

12-month minimum. No upfront purchase. One robot replaces 3–4 workers — cheaper than the labour it replaces from month one.

## Target Market

- **Beachhead:** Dutch tomato/cucumber/pepper operators, 3–10 ha, Westland/Lansingerland
- **SAM (Netherlands):** €167–223M/year
- **SOM Year 3:** €8–14M ARR

## The One Number That Matters

A human picks ~150–200 fruit/hr. The robot must sustain ~450–600 fruit/hr to replace 3 workers. **This is what the prototype has to prove before anything else matters.**
